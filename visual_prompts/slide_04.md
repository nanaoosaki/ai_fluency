# Slide 04 Generation Prompt: The Context System

## Purpose

Define the method as a five-stage working contract.

## Exact Text To Render

Render exactly this text, with correct spelling and punctuation, and no additional words:

```text
THE METHOD

A context system turns documentation into a working contract.

Sources       Synthesis       Protocol       Execution       Feedback
notes         meaning         decisions      analysis        review
schemas       structure       constraints    drafts          corrections
tests         priorities      rules          code            updated context

Documentation is not storage.
It is the working contract between human judgment and AI capability.
```

## Visual Direction

Create a calm, precise 16:9 systems infographic. Five horizontally connected stages should be the primary evidence object, marked by understated dots and a continuous line. Use warm paper, ink-black typography, stone hairlines, and muted forest emphasis. The design should feel like documentation architecture, not a corporate process chart.

## Layout

- Kicker and headline at top.
- Five equally readable stages across the center, with their three supporting lines.
- Two-line conclusion near the bottom.

## Requirements

- Full finished slide.
- All five headings and supporting terms must be readable and spelled correctly.
- No decorative icons, extra stages, logos, watermark, or pseudo-text.
