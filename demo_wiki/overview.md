# Agent Notes Demo Overview

## Purpose

Evaluate whether an AI-assisted workflow can turn unstructured project notes into a useful, reviewable summary without treating model output as final truth.

## Current System

1. A small note sample is supplied as source material.
2. AI produces candidate structured fields and a short summary.
3. A human reviews the result before it is used downstream.
4. Review findings are written back into operational documentation.

## Current Focus

The current iteration is testing which evidence fields are stable enough to extract consistently and which require additional clarification or human judgment.
